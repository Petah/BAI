<h1>Balanced Annihiliation AI (BAI)</h1>

<h2>Features:</h2>
<ul>
    <li>Multithreading (including caching of most of Spring's data so it can be accessed on other threads)</li>
    <li>GUI visualisation (including height maps, threat maps, target maps, movement islands, etc)</li>
    <li>Real time adjustable JavaScript interface (WIP)</li>
    <li>Team cooperation</li>
    <li>Multiple strategies (porc, rush, eco, etc)</li>
    <li>Resource sharing</li>
</ul>

<h2>Commands:</h2>
<ul>
    <li>-showgui, -hidegui</li>
    <li>-allcheat (enables cheating, currently it shouldn't affect anything)</li>
    <li>-hi (says hello world, to make sure the AI is still there)</li>
    <li>-dir (display some debug information about the directories)</li>
    <li>-savesettings (save the current options to a file)</li>
</ul>

<p>
If you are going to use the GUI I recommend you run spring in window mode.<br />
The AI is specifically designed for BA only.<br />
